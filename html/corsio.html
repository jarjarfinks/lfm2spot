About This Tool

This website is a project I took up to make tools to bridge the gap of mobile Spotify and Last.fm. It was created to make it easy to keep an ongoing Spotify Playlist for the purpose of psuedo-mobile Last.fm radio.


I found that the existing Last.fm Radio was lacking. Since 2014, Last.fm has removed mobile radio. It is one of the main reasons many Last.fm users had been willing to subscribe for the measly payment of $3/month. Unfortunately, they removed it, replaced it with YouTube, then replaced it with a "new" user interface with Spotify and YouTube. The "new" Last.fm has changed many times, having varying iterations, all lacking the ability to really be used on a mobile device.


As an avid Last.fm user, I had noticed there were tons of "workarounds" that could be done. Most of these involved exporting scrobbles in the hope you'd end up with similar recommendations on Spotify. These were often very involved and required at least some coding experience.


So, I started working on a tool to take input from your ever changing Last.fm library and generate an always changing Spotify playlist generator.
